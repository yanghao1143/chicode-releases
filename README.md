# ChiCode Desktop

> AI 编程助手桌面版 —— 安装即用，零门槛

## 下载安装

前往 [**Releases 页面**](../../releases/latest) 下载最新版安装包。

| 平台 | 文件格式 | 说明 |
|------|----------|------|
| Windows x64 | `.msi` | 双击安装即可使用 |

> macOS 和 Linux 版本即将推出

## 激活说明

软件安装后需要激活才能使用，请先获取激活密钥：

1. 访问 [**claude.chiddns.com**](https://claude.chiddns.com) 获取 API Key
2. 安装并启动 ChiCode Desktop
3. 在激活页面输入获取到的 Key
4. 激活成功，开始使用

> 没有 Key 无法使用软件，请先前往 [claude.chiddns.com](https://claude.chiddns.com) 获取。

## 安装步骤

1. 点击上方链接，下载 `.msi` 文件
2. 双击运行安装程序
3. 启动 ChiCode Desktop
4. 输入从 [claude.chiddns.com](https://claude.chiddns.com) 获取的 Key 完成激活
5. 开始使用

## 功能特性

- **零配置** - 无需手动安装 CLI 或配置环境变量
- **离线可用** - 安装包内置完整运行环境，弱网也能用
- **内置 AI 工具** - 集成 OpenClaw、Codex、Claude Code、Gemini CLI
- **自动更新** - 检测到新版本一键升级

## 系统要求

- Windows 10/11 (x64)
- 约 600MB 磁盘空间

## 常见问题

**Q: 如何获取激活 Key？**
A: 访问 [claude.chiddns.com](https://claude.chiddns.com) 注册并获取。

**Q: 安装时 Windows 弹出安全警告怎么办？**
A: 当前为预览版，暂未进行代码签名。点击「更多信息」→「仍要运行」即可。

**Q: 首次启动很慢？**
A: 首次启动需要初始化运行环境，请耐心等待 1-2 分钟。
